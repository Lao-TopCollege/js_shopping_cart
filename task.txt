// ຟັງຊັ້ນເພີ່ມລາຍການສິນຄ້າ 
function addCartItem(selectedShirt){
     let cartItem = JSON.parse(localStorage.getItem('cart-item'));

    if(cartItem != null) {
        if(cartItem[selectedShirt.id] == undefined){
            selectedShirt.qty = 1;
            cartItem = {
                ...cartItem,
                [selectedShirt.id]:selectedShirt,
            }
        }
        cartItem[selectedShirt.id].qty += 1;
    } else {
        selectedShirt.qty = 1;
        cartItem = {
            [selectedShirt.id]:selectedShirt
        }
    }
    localStorage.setItem('cart-item', JSON.stringify(cartItem));
}
